# Initial Steps

```
npm install
```

# Running transpiling

```
node_modules/.bin/babel code/ -d build/
```

# Running code (just a sample)

```
node build/11.default-export-function-2.js
```

# Testing with Babel and other ES versions using 'presets'

```
npm install --save-dev babel-preset-es2020
```

OR

```
npm install --save-dev babel-preset-es2020
```
